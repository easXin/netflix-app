# Netflix Clone

An archived early frontend project built to practice React component development, responsive UI layout, API-driven content rendering, and Firebase Hosting deployment. The application recreates a Netflix-style browsing experience with reusable React components and a media-focused user interface.

## Live Demo

https://netflix-clone-3ace6.web.app/

## Project Overview

This project was created as an early React practice project to strengthen frontend development fundamentals. It focuses on building a streaming-platform-style interface, organizing reusable components, rendering media content dynamically, and deploying a React application to Firebase Hosting.

## Tech Stack

- React
- JavaScript
- HTML
- CSS
- Firebase Hosting

## Features

- Netflix-style responsive frontend layout
- Reusable React components for page sections and media rows
- API-driven content rendering for movie and media data
- Media-focused user interface inspired by modern streaming platforms
- Public deployment using Firebase Hosting

## Project Status

This is an archived early personal project. It is kept as part of my project history to show my early frontend development practice and progression toward more advanced full-stack, enterprise, and cloud-based projects.
